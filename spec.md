# Excel PowerPoint Learning Game - Interactive Simulation - SATOR Blood Red Theme

## Overview
An engaging 2D learning game with seven levels teaching Excel and PowerPoint skills through hands-on interactive simulations using scenario-based problem statements. Users perform real Excel and PowerPoint actions within realistic business contexts rather than answering multiple choice questions. The application features a dramatic blood red visual theme with dark contrasting backgrounds, red glowing shadows, neon effects, and the prominent display of "SATOR" branding with metallic styling and luminous outlines.

## Game Structure
- Seven distinct levels with educational content
- Ten tasks per level for comprehensive learning
- Progressive difficulty across levels
- Interactive Excel and PowerPoint simulation interfaces with contextual datasets
- Two-attempt system per task with lives mechanism
- Enhanced audio-visual feedback system
- Animated visual timers with red pulsing effects and extended duration
- Dynamic score tracking with red glowing counters
- Animated progress indicators and completion badges
- Sequential task progression requiring completion before advancing

## Scenario-Based Task Design System
- **Real-World Business Contexts**: Each task presents authentic workplace scenarios (e.g., "Your manager needs the quarterly sales summary for the board meeting", "Calculate profit margins for the new product line")
- **Narrative Problem Statements**: Tasks begin with contextual storytelling that explains the business need and expected outcome
- **Structured Data Tables**: Instructions bar displays both the narrative description and a formatted data table that serves as the dataset for Excel simulation
- **Contextual Learning**: Users solve practical business problems using Excel and PowerPoint skills within realistic workplace situations
- **Immersive Storytelling**: Clear but engaging business-like narratives that make learning relevant and applicable
- **Dynamic Contexts**: Different scenarios across all 7 levels and 10 tasks, covering various business functions (sales analysis, employee performance, financial reporting, project management, etc.)
